Esse é um projeto CRUD para a BetaBit

Aqui está sendo desenvolvido um CRUD para cadastro de usuários

Para acessa-lo, basta usar o localhost:3000/users/

Todos os paths disponíveis serão disponibilizados abaixo:

  • localhost:3000/users: Retorna todos os usuários
  • localhost:3000/users/:id: Retorna um usuário passando como parâmetro o ID.
  • localhost:3000/users/create: Cria um novo usuário
  • localhost:3000/users/update/:id: Atualiza o registro de um usuário passando como parâmetro o seu ID
  • localhost:3000/users/delete/:id: Deleta um usuário passando como parâmetro o seu ID
